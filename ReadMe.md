# Humphrey Come Home

## Overview
Combine nearby shelters into one searchable site and use photo-based matching to show pets that look similar to your lost pet.

## Tech Stack
- Backened (Django)
- Frontend (Django + ReactJS)
- Environment (Docker Compose)
- Database (Supabase w/ PostGIS)
- Map Integration (Google Maps API)
